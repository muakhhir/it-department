## 📌 Overview

This project presents an Intrusion Detection System (IDS) using Machine Learning to detect cyber threats in real time. The system is implemented as IDS Shield, an AI-powered platform that uses multiple machine learning models and intelligent pattern analysis to improve detection accuracy and reliability.

Unlike traditional IDS systems that rely only on rule-based detection, IDS Shield leverages Machine Learning techniques to identify both known attacks and unknown (zero-day) threats.

---

## 🚀 Features

### 🔍 Multi-Model Detection
- Random Forest  
- Support Vector Machine (SVM)  
- Decision Tree  
- Naive Bayes  

### ⚡ Real-Time Threat Analysis
Detects malicious activity instantly from logs  

### 📊 Risk Score Generation
Provides numerical risk scores for better understanding  

### 📈 Interactive Dashboard
Displays results using charts, graphs, and visual analytics  

### 🤖 AI Threat Assistant
Explains detected threats in simple human-readable language  

### 🔔 Alert System
Generates alerts for suspicious and malicious activities  

---

## 🧠 Working Principle

1. User inputs logs or system data  
2. Data is preprocessed and cleaned  
3. Machine Learning models analyze the input  
4. Risk scores are generated  
5. Best-performing model is selected  
6. Results are displayed on the dashboard
